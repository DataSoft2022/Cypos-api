#+title: CyPOS Integration API

the cypos is a frappe app that Integrates ERPNext sales invoice with CyPOS

* API Reference

** Getting started

   - BaseUrl :  the website url on which the app is installed.
   - Header: Authorization api key and secret generated by Frappe Framework,
     - Authorization: token api_key:api_secret
   
** Endpoints

   [[#][POST /api/method/cypos.api.new_sales_invoice]]

   - General 
     - create new Sales Invoice and add *items* to the invoice,
       if the item doesn't exist it creates new one.

   - Sample:
     #+BEGIN_SRC shell
curl --location --request POST 'http://develop.test:8000/api/method/cypos.api.new_sales_invoice' \
--header 'Authorization: token b15d18f73412d7a:5e07497a73a5340' \
--header 'Content-Type: application/json' \
--data-raw '{
    "data": {
        "customer": "Ahmed",
        "territory": "Egypt",
        "items": [
            {                
                "item_code": "7-1.5L",
                "item_name": "7Up Mega 1",
                "item_group": "All Item Groups",
                "qty": 3,
                "rate": 200
            },
            {                
                "item_code": "BUR-S",
                "item_name": "Beef Burger",
                "item_group": "All Item Groups",
                "qty": 1,
                "rate": 75
            }
        ]
    }
}'
     #+END_SRC

     - response
       #+BEGIN_SRC shell
       {
           "message": {
	      "success": true
	      }
       }
       #+END_SRC
